How to configure environment variables in bash?

Here are the **simple ways to configure environment variables in Bash**:

***

## ✅ 1. Temporary (current terminal session)

```bash
export NAME=Nishant
```

Use it:

```bash
echo $NAME
```

👉 Works only in the current terminal session

***

## ✅ 2. For a single command

```bash
NAME=Nishant bash script.sh
```

👉 Available only for that command

***

## ✅ 3. Permanent (for user)

Add to your `~/.bashrc` or `~/.bash_profile`:

```bash
export NAME=Nishant
```

Then apply changes:

```bash
source ~/.bashrc
```

👉 Available every time you open the terminal

***

## ✅ 4. System-wide (for all users)

Edit `/etc/environment`:

```bash
NAME=Nishant
```

👉 Requires restart or re-login

***

## ✅ 5. Using a `.env` file

**.env file:**

    NAME=Nishant

**Load in script:**

```bash
source .env
echo $NAME
```

***

## 🎯 Interview Answer (Short)

"We can configure environment variables in Bash using the `export` command for temporary usage, define them in `.bashrc` or `.bash_profile` for permanent use, or store them in `.env` files and load them in scripts. This helps in externalizing configuration and making applications environment-independent."
