What is cron job? How do you use cron job in python for administrative task?

### ✅ What is a Cron Job?

A **cron job** is a time-based scheduler in Unix/Linux systems that automatically runs scripts or commands at specified intervals (e.g., every minute, hour, day, or week).

👉 In simple terms:

> A cron job is used to **automate repetitive tasks** like backups, cleanup, sending emails, or running scripts without manual execution.

***

### ✅ How Cron Job Works

* It uses a configuration file called **crontab**
* You define:
  * **When to run** (schedule)
  * **What to run** (command/script)

**Example:**

```
0 2 * * * /path/to/script.sh
```

➡ Runs every day at 2 AM

***

# ✅ Using Cron Job in Python for Administrative Tasks

Python itself doesn’t have cron built-in, but you can:

1. **Write a Python script**
2. **Schedule it using cron**

***

## ✅ Step 1: Create a Python Script

Example: `cleanup.py`

```python
import os
import datetime

def cleanup_logs():
    log_dir = "/var/log/myapp/"
    now = datetime.datetime.now()

    for file in os.listdir(log_dir):
        file_path = os.path.join(log_dir, file)
        if os.path.isfile(file_path):
            file_age = now - datetime.datetime.fromtimestamp(os.path.getmtime(file_path))

            # Delete files older than 7 days
            if file_age.days > 7:
                os.remove(file_path)
                print(f"Deleted: {file_path}")

if __name__ == "__main__":
    cleanup_logs()
```

👉 This script deletes old log files (example of admin task)

***

## ✅ Step 2: Schedule it using Cron

Open crontab:

```
crontab -e
```

Add entry:

```
0 1 * * * /usr/bin/python3 /path/to/cleanup.py
```

👉 This runs the Python script every day at 1 AM

***

## ✅ Common Use Cases in Python + Cron

* Log cleanup
* Database backups
* Sending scheduled email reports
* Health checks / monitoring scripts
* Data processing jobs
* Cache clearing

***

## ✅ Best Practices

* ✅ Use full path for Python (`/usr/bin/python3`)
* ✅ Add logging to track execution:
  ```
  0 1 * * * /usr/bin/python3 cleanup.py >> log.txt 2>&1
  ```
* ✅ Handle exceptions in script
* ✅ Test script manually before scheduling

***

# ✅ Interview Summary (Short Answer)

A cron job is a scheduler used to automate tasks at specific time intervals in Unix/Linux systems. In Python, we typically write a script for an administrative task such as log cleanup or data backup, and then schedule it using cron by adding an entry in the crontab file. This allows the Python script to run automatically at defined intervals without manual intervention, making it very useful for routine maintenance and background jobs.
