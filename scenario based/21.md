Which command is used to access a container terminal?

**“The command used to access a container terminal in Kubernetes is `kubectl exec`.”**

### ✅ Example:

```bash
kubectl exec -it <pod-name> -- /bin/sh
```

or (if bash is available):

```bash
kubectl exec -it <pod-name> -- /bin/bash
```

### 💡 Explanation:

*   `-i` → interactive mode
*   `-t` → allocates a terminal
*   `<pod-name>` → name of the pod
*   `-- /bin/sh` → opens a shell inside the container

👉 If the pod has multiple containers:

```bash
kubectl exec -it <pod-name> -c <container-name> -- /bin/sh
```

***

### 🎯 Interview Tip:

**“We use `kubectl exec -it` to get an interactive shell inside a running container for debugging and inspection.”**
